## Streaming Pipeline
- Kick Chat WebSocket (real-time, public, no API key needed)
- Chat messages → Pluggable Reasoning Engine → generates response text
- Response → TTS (multiple providers: ElevenLabs, Fish Audio, macOS say)
- Audio → Avatar rendering (Live2D or Anam Cara 3)
- Avatar → OBS Browser Source (localhost:3939)
- OBS → RTMPS → Kick/Twitch
- Full pipeline latency: ~200ms with Anam Cara 3

## Avatar System — Anam Cara 3
- Photorealistic AI face generated from a single photo (one-shot)
- 180ms latency, WebRTC stream
- Micro-expressions, blinking, natural idle animations
- SDK: @anam-ai/js-sdk
- Key methods: createClient(sessionToken), streamToVideoElement(), sendUserMessage(text), stopStreaming()
- Session token flow: API key → POST /v1/auth/session-token → sessionToken → createClient()
- Can disable microphone (disableInputAudio: true) — text-only input for chat-driven streaming
- Supports custom system prompts per session

## Character Creator
- Upload photo → Gemini Vision analysis → IdentityBlock/BodyBlock → Gemini Imagen → turnaround + portrait
- Supabase backend (PostgreSQL + Realtime for progress)
- Supports styles: anime, realistic, pixel, cyberpunk, watercolor, comic, 3D, fantasy
- Frontend: Next.js 14 app (apps/character-web, port 3001)
- Package: packages/character-creator
